# Expense Tracker Frontend

Frontend do projeto de controle financeiro feito com Next.js, React, TypeScript e Tailwind CSS.

## Scripts

- `npm run dev` inicia o ambiente de desenvolvimento.
- `npm run build` gera o build de produção.
- `npm run start` executa a aplicação em produção.
- `npm run lint` executa a checagem de lint.

## Estrutura

- `src/app` contém as rotas do App Router.
- `src/components` concentra as telas e componentes reutilizáveis.
- `src/services` concentra integrações com a API.
